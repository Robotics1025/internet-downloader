# Download Manager

Open-source, local-only desktop download manager. Electron + React frontend, Python FastAPI backend, SQLite + Alembic storage, Manifest V3 browser extension.

## Status

**Phase 1 — Foundation** (in progress). See `docs/superpowers/specs/` for current spec and `docs/superpowers/plans/` for the active implementation plan.

## License

Open source. No telemetry, no cloud sync, no remote management — runs entirely on the user's machine.
